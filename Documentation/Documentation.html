BridgeMail Module

* Important *

Due to limitations in the templating capabilities of Visual Studio, you will need to make some minor changes before you begin development.

  • Edit the Project properties by double-clicking on the Properties folder in Solution Explorer
  • On the Application tab:
    • Make sure the Assembly name textbox is set to "Gooddogs.Modules.Dnn.BridgeMail".
    • Make sure the Default namespace is set to "Gooddogs.Modules.Dnn.BridgeMail".
  • On the Web tab:
    • Under 'Start Action', make sure to set your Start URL to the correct site (ex: http://localhost/{yourDnnSite}).
    • Under 'Servers', make sure to "Use IIS Web Server" with a project url of http://localhost/{yourDnnSite}/desktopmodules/BridgeMail and an Override application root URL of http://localhost/{yourDnnSite}/
